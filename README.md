# 사용법 
- 위치조절 : 좌클릭 + 드래그
- 크기조절 : 우클릭 + 드래그
- 사용종료 : 중클릭
- 사진교체 : 드래그 앤 드롭
- 기본사진 : 같은 파일위치 내 image.jpg

# 설명 
- App.py : 메모장 등으로 script 확인
- App.pyw: python으로 프로그램 실행(pyw : 콘솔창 숨기고 실행)
- App.exe: 실행파일

# exe로 저장 
- cmd로 App.py까지 이동
- pyinstaller -w -F Image-Sticker.py
